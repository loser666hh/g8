# 我破產了 💃https://loser666hh.github.io/g8/

一個互動式網頁遊戲，讓你用鍵盤控制一個搞笑的舞者人物跳舞！

## 🎮 功能特色

- **鍵盤控制舞蹈** - 使用 Q、W、A、S、Z、X、C、V 八個按鍵控制舞者的手臂和雙腿
- **實時動畫** - 流暢的動畫效果和視覺反饋
- **隨機舞蹈** - 點擊按鈕生成搞笑的隨機舞蹈動作
- **重置功能** - 一鍵重置舞者回到初始狀態
- **響應式設計** - 支援各種屏幕尺寸（桌面、平板、手機）
- **漸層背景** - 優美的紫色漸層背景和毛玻璃效果

## 🕹️ 控制方式

| 按鍵 | 動作 |
|------|------|
| **Q** | 左手上抬 |
| **W** | 右手上抬 |
| **A** | 左手下擺 |
| **S** | 右手下擺 |
| **Z** | 左腳踩地 |
| **X** | 右腳踩地 |
| **C** | 左腳抬起 |
| **V** | 右腳抬起 |

### 💡 小貼士
同時按多個鍵可以做出更搞笑的組合動作！例如同時按 Q+W+Z+X 可以做出四肢並用的瘋狂舞蹈。

## 🚀 快速開始

### 在線遊玩
1. 打開 [GitHub Pages 鏈接]（如已部署）
2. 或直接在本地打開 `index.html` 文件

```bash
